# crassuspiscis.github.io

## Отчет

Работу я разделил на части:

1) Создание общей структуры
2) Внутреннее оформление начальных страниц
3) Внутреннее оформление страниц с лабораторными работами
4) Попытка выпуска сайта на общее обозрение
5) Cоздание красивого оформления

## Структура:
  - Гланвая страница
  - Обо мне
  - Лабораторные работы

## Лабораторные работы:
Использовались самые интересные (по моему мнению), в которых раскрываются определенные навыки

## Выбранная тема:
Я выбирал из нескольких тем: windmill, dracula, readthedocs, material

В них мне не понравилась стандартная система навигации между страницами - они были похоже сайт для документаций (windmill) или для организации детских праздников (dracula),
что не является эстетически приемлемым для сайта-портфолио, которое обязано содержать характер серьезного
настроя к работе

Для меня самой приятной оказалась тема material, так как устраивала система навигации на страницах, а внешний вид
был допустим для оформления под сайт-портфолио

## Некоторые особенности

1) Переключение между светлой/темной темам, для большего комфорта глаз
2) Кнопка "К началу", которая позволяет не перелистывать страницу самостоятельно, а перейти к навигации по нажатию кнопки
3) Интерактивные кнопки на странице "Сотрудничество", некий "психологический трюк", так как заинтересованому гостю сайта не придется рассматривать длинные ссылки, достаточно также просто нажать на кнопку чтобы связаться
