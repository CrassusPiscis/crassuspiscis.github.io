    Лабораторная работа №5

    Цель

    Разработать функцию для генерации бинарного дерева с заданными параметрами и реализовать визуализацию структуры дерева в виде вложенного словаря.

    Задание

    Написать функцию gen_bin_tree, которая принимает следующие параметры: - root — значение корня дерева (по моему варианту: 1) - height — высота дерева (количество поколений) (по моему варианту: 5) - left_branch — функция для генерации левого потомка (по моему варианту: x * 2) - right_branch — функция для генерации правого потомка (по моему варианту: x + 3)

    Функция должна возвращать бинарное дерево в виде вложенного словаря, где каждый узел представлен как {значение: [левый_потомок, правый_потомок]}.

    Код программы

    def gen_bin_tree(root, height, left_branch=lambda x: x * 2, right_branch=lambda x: x + 3) -> dict[str, any]:
    '''
    :param height: Задаёт количество "поколений" дерева, включая "поколение" корня
    :param root: значение корня
    :param left_branch: значение для левого "наследника"
    :param right_branch: значение для правого "наследника"
    :return: готовое дерево
    '''
    # рассматриваем отдельный случай, когда количество поколений равно 1
    if height == 1:
        return {f'{root}': []}

    # генерация значений всех поколений дерева
    tree = [[root]]
    for i in range(height-1):
        last_level = tree[-1]
        new_level = []
        for i in last_level:
            tree_level_left = left_branch(i)
            tree_level_right = right_branch(i)
            new_level += [tree_level_left, tree_level_right]
        tree.append(new_level)

    # функция генерации словаря
    def dict_gen(key, equal) -> dict[any, any]:
        return {f'{key}': equal}

    # Обратная сортировка дерева для более удобной реализации
    tree.sort(reverse=True, key=len)

    # Создание "словарного" вида последних элементов
    end_of_tree = []
    for i in range(len(tree[0])):
        end_of_tree.append(dict_gen(tree[0][i], []))
    tree[0] = end_of_tree

    # занесение всех элементов поколений в конструкцию дерева "словарно-списочного" вида
    for i in range(1, len(tree)):
        new_level = []
        for j in range(len(tree[i])):
            new_level += [dict_gen(tree[i][j], [tree[i-1][2*j], tree[i-1][2*j+1]])]
        tree[i] = new_level

    # создание готового дерева, берется корень и остальная сформированная конструкция дерева
    ready_tree = dict_gen(root, tree[-2])

    return ready_tree

print(gen_bin_tree(1, 5))

    Вывод

    В ходе выполнения лабораторной работы была реализована функция gen_bin_tree, которая генерирует бинарное дерево заданной высоты.

    Основные результаты:

    1. Функция принимает параметры:
    2. root — значение корня дерева
    3. height — высота дерева (количество поколений)

    4. left_branch и right_branch — функции для вычисления значений потомков

    5. Особенности реализации:

    6. Функции-аргументы по умолчанию: x * 2 для левой ветви и x + 3 для правой
    7. Возвращаемое значение — вложенный словарь вида {значение: [левый_потомок, правый_потомок]}

    8. Обработка граничного случая при height = 1

    9. Принцип работы алгоритма:

    10. Сначала генерируются значения всех узлов по уровням
    11. Затем выполняется преобразование линейной структуры во вложенные словари

    12. Обратная сортировка уровней упрощает процесс вложения

    13. Гибкость использования:

    14. Возможность подставить любые функции для генерации потомков
    15. Функция может применяться для моделирования различных древовидных структур

    Таким образом, реализована универсальная функция для генерации бинарного дерева с возможностью кастомизации правил ветвления.
